# Инструмент для работы с 3D-моделями в вебе. Помогает автоматически центрировать модели и показывать их точные границы, даже если модели были созданы неправильно или "сырые".

Демонстрация
https://anastasy-ya.github.io/threejs_project/

## Особенности
Работает с неподготовленными моделями (с неприменёнными трансформациями).
Автоматически центрирует меш-объекты по их геометрическому центру.
Создаёт и обновляет визуальные ограничивающие боксы (BoxHelper) для каждого меша.
Позволяет исключать определённые мешы из обработки.
Подходит для сложных иерархических моделей.

## Технологии
Three.js — основной движок.
Webpack — сборка.
